## Image Gallery

A single page application that uses Unsplash api to fetch images on random, or by search query.

#### Technologies used:
- [Next.Js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com)
- [React Query](https://react-query-v3.tanstack.com)
